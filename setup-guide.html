🚂 OpenRailway on Google Map

此專案展示如何將 OpenRailway 鐵路路網資料套疊到 Google Maps 上，讓您可以同時查看詳細的鐵路網路與 Google 地圖。

✨ 功能特色

🗺️ 雙層地圖

Google Maps 作為底圖，OpenRailway 資料以透明圖層疊加

🔄 圖層切換

可隨時切換顯示或隱藏鐵路圖層

🇹🇼 台灣聚焦

預設中心點在台灣，方便查看本地鐵路網

📱 響應式設計

支援桌面與行動裝置瀏覽

🚀 快速開始

1 取得 Google Maps API Key
  • 前往 Google Cloud Console
  • 建立新專案或選擇現有專案
  • 啟用 "Maps JavaScript API"
  • 在憑證頁面建立 API 金鑰
2 設定 API Key

編輯 index.html，將 YOUR_API_KEY 替換為您的 API 金鑰：

<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap&v=weekly" async defer></script>
3 開啟網頁

使用瀏覽器開啟 index.html

建議使用本地伺服器：

# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server
✅ 無需 API Key 的測試版本

如果您暫時沒有 Google Maps API Key，可以使用 demo-leaflet.html

此版本使用開源的 Leaflet 地圖庫搭配 OpenStreetMap，無需 API Key 即可運作。

🏗️ 技術架構

核心技術

  • Google Maps JavaScript API - 提供互動式地圖介面
  • OpenRailwayMap Tiles - 提供鐵路路網圖磚資料
  • ImageMapType - 將外部圖磚整合到 Google Maps

📋 檔案說明

  • index.html - 主要的 Google Maps 版本
  • demo-leaflet.html - 使用 Leaflet 的測試版本（無需 API Key）

  • README.md - 專案說明文件
